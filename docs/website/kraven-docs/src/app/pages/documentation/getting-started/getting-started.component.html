Getting Started with Kraven UI

v1.0.7

Introduction

Kraven UI is a ridiculously good-looking, highly readable, and customizable documentation UI for OpenAPI specifications generated by springdoc-openapi. It combines the elegant readability of Redoc with the interactive playground capabilities of Swagger UI, but with way more style and fewer existential crises.

More than just API documentation, Kraven UI is a complete development and debugging toolkit that brings together API testing, Feign client exploration, Kafka management, and application monitoring in one unified interface. It's designed to make your development workflow smoother, debugging less painful, and your life as a developer significantly better.

Installation

Maven

Add the following dependency to your Spring Boot application's pom.xml:

<dependency>
    <groupId>io.github.rohitect</groupId>
    <artifactId>kraven-ui</artifactId>
    <version>1.0.7</version>
</dependency>

Ensure SpringDoc OpenAPI is Configured

Make sure you have SpringDoc OpenAPI configured:

<dependency>
    <groupId>org.springdoc</groupId>
    <artifactId>springdoc-openapi-starter-webmvc-ui</artifactId>
    <version>2.3.0</version>
</dependency>

Basic Configuration

Add configuration to your application.properties or application.yml file:

Properties Format

# Kraven UI Configuration
kraven.ui.enabled=true
# Note: The UI path is hardcoded to /kraven/ui and API path to /kraven/api

# Theme configuration
kraven.ui.theme.dark-primary-color=#6c5ce7
kraven.ui.theme.dark-secondary-color=#00b894
kraven.ui.theme.dark-background-color=#1a1b2e
kraven.ui.theme.light-primary-color=#1976d2
kraven.ui.theme.light-secondary-color=#424242
kraven.ui.theme.light-background-color=#ffffff
kraven.ui.theme.default-theme=dark

# SpringDoc OpenAPI Configuration
springdoc.api-docs.path=/v3/api-docs
springdoc.swagger-ui.enabled=false  # Ditch Swagger UI, you've upgraded

YAML Format

kraven:
  ui:
    enabled: true
    # Note: The UI path is hardcoded to /kraven/ui and API path to /kraven/api

    theme:
      # Dark theme specific colors
      dark-primary-color: "#6c5ce7"
      dark-secondary-color: "#00b894"
      dark-background-color: "#1a1b2e"

      # Light theme specific colors
      light-primary-color: "#1976d2"
      light-secondary-color: "#424242"
      light-background-color: "#ffffff"

      default-theme: dark

springdoc:
  api-docs:
    path: /v3/api-docs
  swagger-ui:
    enabled: false  # Ditch Swagger UI, you've upgraded

Environment Variables

You can also configure Kraven UI using environment variables with JSON format:

export KRAVEN_UI_CONFIG='{{ "{" }}"theme":{{ "{" }}"darkPrimaryColor":"#ff5722"{{ "}" }}{{ "}" }}'

Running Your Application

Once you've added the dependency and configuration, simply run your Spring Boot application. Kraven UI will be available at /kraven/ui. 

./mvnw spring-boot:run

Navigate to http://localhost:8080/kraven/ui in your browser to access Kraven UI.
