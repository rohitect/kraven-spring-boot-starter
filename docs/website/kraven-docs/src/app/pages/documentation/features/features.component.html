API Documentation

The API Documentation UI is where Kraven UI really shines. It combines the elegant readability of Redoc with the interactive capabilities of Swagger UI, but with a much more modern and customizable interface.

Key Features:

  • Three-Pane Layout - Navigation sidebar, API details, and request/response panels
  • Dual-Mode Interface - Switch between Documentation and Playground modes
  • Syntax Highlighting - Beautiful code highlighting for JSON and other formats
  • Markdown Support - Rich formatting for API descriptions
  • Schema Visualization - Interactive schema diagrams
  • Enhanced Search - Find endpoints by path, description, or parameters with tag-based filtering
  • Favorites System - Bookmark frequently used endpoints for quick access
  • Dark/Light Themes - Choose your preferred visual style
  • Responsive Design - Works on desktop and mobile devices
  • Context Menu - Right-click on tabs for quick actions

Documentation Experience:

  • Comprehensive Details - Complete information about each endpoint
  • Tag-Based Organization - Endpoints grouped by logical categories
  • Parameter Documentation - Detailed descriptions of all parameters
  • Response Examples - Sample responses for different status codes
  • Schema References - Links between related data models

API Playground

The API Playground allows you to test your API endpoints directly from the documentation. It provides a user-friendly interface for constructing requests and viewing responses, with enhanced features for both learning and testing.

Key Features:

  • Dual-Mode Interface - Switch between Documentation mode (for learning) and Playground mode (for testing)
  • Request Builder - Easily construct API requests with a form-based interface
  • Parameter Validation - Validate request parameters before sending
  • Authentication Support - Test authenticated endpoints with various auth methods
  • Response Visualization - See formatted responses with syntax highlighting
  • Request History - Keep track of previous requests for quick reference
  • Favorites System - Bookmark frequently used endpoints for quick access across sessions
  • Tag-Based Search - Find endpoints using multiple search tags with context highlighting
  • Context Menu - Right-click on tabs for quick actions like closing, favoriting, or copying as cURL
  • Copy as cURL - Generate cURL commands for your requests
  • Download Response - Save API responses for later use

Documentation Mode:

  • Focused Reading - Clean interface for learning about APIs without execution distractions
  • Enhanced Search - Find endpoints by path, description, or parameters with tag-based filtering
  • Comprehensive Details - View complete API documentation with all parameters and response types
  • One-Click Testing - Easily switch to Playground mode to test any endpoint

Playground Mode:

  • Interactive Testing - Try out API endpoints with a powerful request builder
  • Sample Request Bodies - Auto-generated examples so you don't have to guess the format
  • Response Inspection - Examine detailed responses with syntax highlighting
  • Request History - Track your API call history for easy reference and reuse

Feign Client Explorer

The Feign Client Explorer automatically discovers and displays all Feign clients in your application, allowing you to explore and test external service integrations.

Key Features:

  • Auto-Discovery - Automatically finds all Feign clients in your application
  • Method Exploration - View and test all methods on each client
  • Parameter Testing - Try out different parameter values
  • Response Visualization - See the actual responses from your external services
  • Client Search - Find that one client among the hundreds you've created
  • Metadata Caching - Optimized performance with metadata caching

Kafka Management

The Kafka Management UI provides comprehensive tools for managing your Kafka clusters, topics, and messages. It's available through the Kafka Plugin.

Key Features:

  • Topic Overview - See all your Kafka topics in one place
  • Message Producer - Send messages to topics without writing code
  • Message Browser - View messages in topics with filtering and pagination
  • Live Streaming - Watch messages arrive in real-time
  • Consumer Groups - Monitor consumer groups and their lag
  • Listener Discovery - Automatically discovers KafkaListener annotations
  • Smart Loading - Features a sarcastic loading animation when refreshing

Application Monitoring

The Application Monitoring features provide insights into your application's performance and health, leveraging Spring Boot Actuator endpoints.

Key Features:

  • Health Checks - Monitor the health of your application and its dependencies
  • JVM Metrics - Track memory usage, garbage collection, and more
  • Thread Dumps - Generate and analyze thread dumps
  • Environment - View application environment and configuration
  • Logging - Adjust log levels at runtime
  • HTTP Metrics - Monitor HTTP request performance
  • Custom Metrics - View application-specific metrics

Actuator Insights

Actuator Insights provides a powerful visualization and monitoring layer for Spring Boot Actuator endpoints, making it easy to monitor and manage your application's health, metrics, configuration, and thread activity.

Key Features:

  • Real-time Visualization - Interactive dashboards for application health, metrics, and system resources
  • Smart Auto-detection - Automatically detects Spring Boot Actuator in your application
  • Comprehensive Thread Analysis - Thread state distribution, deadlock detection, and thread pool insights
  • Advanced Environment Visualization - Property source exploration with search highlighting
  • Historical Data Analysis - Track metrics over time with trend visualization
  • Customizable Dashboard - Configure views and auto-refresh intervals

Tab-based Interface:

  • Health & Status - Monitor application health with real-time updates
  • Metrics & Performance - Track key performance indicators with historical trends
  • Environment & Config - Explore configuration with advanced search and filtering
  • Beans & Dependencies - Visualize application structure and dependencies
  • Thread Analysis - Comprehensive thread diagnostics and visualization
  • Conditions Report - Analyze Spring Boot auto-configuration conditions
  • Application Logs - View and analyze application logs with advanced filtering

Thread Analysis Features:

  • Thread State Visualization - Color-coded thread state distribution
  • Deadlock Detection - Identify circular dependencies between threads
  • Lock Contention Analysis - Find locks with multiple waiting threads
  • Thread Pool Analysis - Group threads by common pools (Tomcat, HikariCP, Kafka, etc.)
  • CPU-Intensive Thread Detection - Identify threads likely consuming CPU resources
  • Memory Leak Indicators - Find potential memory leak indicators
  • PDF Report Generation - Create shareable thread analysis reports

Environment Tab Features:

  • Advanced Search - Search across all property sources with real-time highlighting
  • Property Source Visualization - Hierarchical view with property counts
  • Origin Tracking - See where each property was defined
  • Active Profiles Display - Visual representation of active Spring profiles
  • Sensitive Value Masking - Configurable masking of sensitive information

Conditions Report Features:

  • Auto-Configuration Analysis - Understand why Spring Boot auto-configurations were applied or skipped
  • Secondary Tabs - Categorized view of positive matches, negative matches, and unconditional classes
  • Real-time Search - Search across all conditions with highlighted results
  • Condition Counts - See counts of matched and unmatched conditions at a glance
  • Fixed Navigation - Secondary tabs remain fixed when scrolling for easy navigation
  • Detailed Messages - View detailed condition evaluation messages for debugging

Application Logs Features:

  • Real-time Log Viewing - Stream application logs directly from the Spring Boot Actuator logfile endpoint
  • Auto-refresh - Configure automatic refresh intervals to see the latest log entries
  • Log Level Filtering - Filter logs by level (ERROR, WARN, INFO, DEBUG, TRACE) with color-coded indicators
  • Text Search & Filtering - Search for specific text with highlighting and filter log entries
  • Download Logs - Download the complete log file for offline analysis
  • Enhanced Log Viewer - Advanced log viewer with search highlighting, dual-direction pagination, and auto-scroll functionality

Configuration

Configure the Actuator Insights plugin using your application's application.yml or environment variables: 

kraven:
  plugins:
    actuator-insights:
      enabled: true
      auto-detect: true
      base-url: "http://localhost:8080"
      context-path: ""
      data-collection:
        interval: 15s
        retention-period: 1h
      endpoints:
        include: "*"
        exclude: "heapdump,shutdown"
      sensitive-data:
        mask-sensitive-values: true
        sensitive-patterns: "password,passwd,secret,credential,token,key,auth,private,access"

Note: The plugin automatically uses the application's actual port from server.port and context path from either server.servlet.context-path or spring.mvc.servlet.path if configured. The base-url and context-path settings are only used as fallbacks.

Spring Boot Actuator Configuration

To get the most out of Actuator Insights, configure Spring Boot Actuator in your application: 

management:
  endpoints:
    web:
      exposure:
        include: "*"
  endpoint:
    health:
      show-details: always
    env:
      show-values: always  # For Spring Boot 3 to show environment values

# To enable the log viewer
logging:
  file:
    name: /path/to/your/application.log

Enhanced Log Viewer

The Actuator Insights plugin includes a powerful log viewer with advanced features:

  • Dual-direction Pagination - Load older logs at the top and newer logs at the bottom
  • Flexible Log Ordering - Toggle between newest-first and oldest-first views
  • Real-time Search Highlighting - Instantly highlights search terms as you type
  • Advanced Filtering - Filter logs by log level (ERROR, WARN, INFO, DEBUG, TRACE)
  • Auto-scroll Functionality - Automatically scrolls to the newest logs when they arrive
  • One-click Download - Download the complete log file with a single click
  • Auto-refresh Capability - Keep logs updated with configurable refresh intervals

Mock Server Plugin

The Mock Server Plugin provides a configurable mock server for integration testing and development, allowing you to mock API endpoints without depending on external services.

Key Features:

  • Configurable Mock Server - Starts on port 11000 (or custom port)
  • JSON Configuration - Define mock endpoints and responses via JSON
  • Multiple Response Variations - Support for multiple response variations per endpoint
  • Runtime Switching - Switch between different mock responses via the UI
  • Request History - Track and visualize request history
  • Response Delay Simulation - Simulate network latency for realistic testing
  • Dynamic Response Generation - Generate responses based on request data
Mock Server Screenshot

Plugin System

Kraven UI features a powerful plugin system that allows you to extend its functionality with custom plugins. The plugin system provides a clean API for registering new features and UI components.

Key Features:

  • Dynamic Registration - Plugins register with the core at runtime
  • UI Integration - Plugins automatically appear in the UI when registered
  • Independent Lifecycle - Start, stop, and manage plugins independently
  • Configurable - Each plugin has its own configuration options
  • API Extensions - Plugins can extend the Kraven UI API
  • Event System - Event-based notifications for plugin actions
